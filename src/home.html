Tecnologie Utilizzate BOOTSTRAP: è un moderno e potente framework che permette di creare pagine web responsive, cioè adattabili a qualsiasi dispositivo: mobile, tablet o desktop. CSS: è un linguaggio usato per programmare la resa grafica di documenti scritti in un linguaggio di markup, come HTML e XML. JAVASCRIPT: linguaggio di programmazione che gli sviluppatori utilizzano per realizzare pagine Web interattive. AWS: acronimo di Amazon Web Services, è una piattaforma di proprietà del gruppo Amazon che offre servizi di cloud computing, elaborazione e distribuzione di contenuti e molto altro, ideali per creare applicazioni sofisticate in modo flessibile, scalabile e affidabile. Nel nostro caso abbiamo installato uno stack LAMP. DOCKER: un software libero progettato per eseguire processi informatici in ambienti isolabili, minimali e facilmente distribuibili chiamati container.
